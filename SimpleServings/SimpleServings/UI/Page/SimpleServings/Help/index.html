2. Adding Recipe

Recipe Details, Ingredients & Nutrients

  1. Enter a Recipe Name, Yield, Portion Size, Recipe View and Contributor
  2. Select Tag(s) for recipe
  3. Add Ingredients Quantity, Measure Unit and Ingredient Name.
  4. Add cooking directions used to prepare the recipe
  5. Click on the browse button and upload the food nutritional excel file and click on Process

NOTE: Ingredients and Directions can be re-ordered by dragging and dropping. Import excel file to populate Nutrients.

Recipe Recommendations, Requirements, Icons & Notes

Select recommendations and requirements by filling the checkbox. When you are finished, click on Save and you will be redirected to the View Recipe page to see your recipe.

Edit Recipe:

  1. CONTROLS- Add Menu Item, Week in Cycle, Validate, Mark As Complete, Submit to Contract, Submit to DFTA
  2. Click on Validate – then click on Add Menu Item to Edit Menu.
  3. If Validation is complete then click on Mark As Complete and Submit your Menu by click on the corresponding link in the Actions Available box [#]

Recipe List:

  1. Select from drop-down to filter recipe list by category
  2. Search recipes by name or keyword
  3. View or edit recipe from list

3. How To Create a Menu

  1. Select Program Type, Meal Type, Cycle and Days of Service then click the Save button.

  1. CONTROLS- you can Filter by Recipe View [1], Search [2] and select the Week in the Cycle [3]
  2. Search for recipe then drag and drop it in the menu on the right
